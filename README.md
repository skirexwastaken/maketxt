# makeTXT

makeTXT is a simple CLI tool that creates .txt files made for archlinux.

## Installation guide

Clone the repository `git clone https://github.com/skirexwastaken/maketxt`
Go to the folder `cd maketxt`
Build the package `makepkg -si`
Now you can use the package using `maketxt`

## Usage

Create an empty file with a specific name `maketxt main`
Create a file with one line `maketxt main Test`
Create a file with multiple lines `maketxt First line/Second line`
